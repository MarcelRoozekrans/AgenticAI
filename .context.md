## Common Tasks

### Adding a New Feature
1. Start with specification: `docs/specs/_template.spec.md`
2. Use prompt: `.github/prompts/feature-implementation.prompt.md`
3. Follow instructions: `.github/instructions/[domain].instructions.md`

### Fixing a Bug
1. Use prompt: `.github/prompts/bug-fix.prompt.md`
2. Add regression test
3. Update `.memory.md` if relevant

### Code Review
1. Use chat mode: `.github/chatmodes/reviewer.chatmode.md`
2. Or prompt: `.github/prompts/code-review.prompt.md`
